# Khaleesi App

## Requerimientos
- [npm](https://nodejs.org/es/download/package-manager/)
- [electron](https://electron.atom.io/)

## Instalación
```bash
  $ git clone https://github.com/LambdaSnippet/khaleesi-desktop
  $ cd khaleesi-desktop
  $ npm install electron -g
```

## Ejecución
```bash
$ electron .
```
